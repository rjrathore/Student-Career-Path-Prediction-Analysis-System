Student Career Path Prediction System

OVERVIEW :
 This is a full-stack web application developed using Flask, MySQL, HTML, and CSS.  
 The system helps students identify suitable career paths based on their academic performance, interests, and skills.

FEATURES :
- Career prediction based on marks and interests
- Rule-based decision system
- Student data stored in MySQL database
- Dashboard to view all student records
- Clean and responsive user interface

TECHNOLOGIES USED :
- Python (Flask)
- MySQL
- HTML
- CSS
